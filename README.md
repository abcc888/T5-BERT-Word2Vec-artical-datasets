# 面向城市规划评论的民意精准分析

## 📌 项目简介
本仓库包含论文所使用的示例数据与代码：

《面向小样本高噪声评论的民意精准洞察：多模型协同语义增强及其在城市规划评论分析微调中的验证》

本研究提出了一种多模型协同语义增强框架，用于提升小样本、高噪声场景下的民意分析效果。

---

## 📊 数据说明

由于数据来源限制，仅公开部分数据样例。

- 原始数据.csv：城市规划评论示例数据
- 字段说明：
  - `text`：用户评论文本
  - `label`：情感或类别标签

- 纠错处理后数据.xlsx：T5模型处理后数据
- 字段说明：
  - `text`：用户评论文本
  - `text-T5`：T5模型处理后数据
  - `label`：情感或类别标签
 
- 隐喻词提取数据.xlsx：BERT模型处理后数据
- 字段说明：
  - `metaphors`：隐喻词
  - `substutute word`：替换词

- 隐喻词提取数据.xlsx：word2vec模型处理后数据
- 字段说明：
  - `text`：用户评论文本
  - `text-word2vec`：word2vec模型处理后的评论数据

完整数据可在合理请求下向通讯作者获取。

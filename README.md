# 面向城市规划评论的民意精准分析

## 📌 项目简介
本仓库包含论文所使用的示例数据与代码：

《面向小样本高噪声评论的民意精准洞察：多模型协同语义增强及其在城市规划评论分析微调中的验证》

本研究提出了一种多模型协同语义增强框架，用于提升小样本、高噪声场景下的民意分析效果。

---

## 📊 数据说明

- 原始评论数据集.xlsx：城市规划评论示例数据
- 字段说明：
  - `text`：用户评论文本

- 人工标注后数据集.xlsx：城市规划评论示例数据
- 字段说明：
  - `text`：用户评论文本
  - `label1`：匹配主题标签
  - `label2`：情感或类别标签

- 文本纠错训练数据集.xlsx：T5模型训练数据集
- 字段说明：
  - `target`：原始用户评论文本
  - `source`：无错字用户评论文本

- 文本纠错测试数据集.xlsx：T5模型测试数据集
- 字段说明：
  - `target`：无错字用户评论文本
  - `source`：原始用户评论文本

- 纠错处理后数据集.xlsx：T5模型处理后数据
- 字段说明：
  - `target`：无错字用户评论文本
  - `source`：原始用户评论文本
  - `text-T5`：T5模型处理后数据
  - `score`：T5模型处理后数据与无错字数据的匹配得分（百分制）
  - `label1`：匹配主题标签
  - `label2`：情感或类别标签
 
- 隐喻词提取数据集.xlsx（示例）：BERT模型处理后得到的隐喻词以及对应的替换词
- 字段说明：
  - `metaphors`：隐喻词
  - `substutute word`：替换词

- 隐喻词替换后数据集.xlsx（示例）：word2vec模型处理后数据集
- 字段说明：
  - `text`：用户评论文本
  - `text-word2vec`：word2vec模型处理后的评论数据

- 原始数据-主题匹配.json：原始数据主题匹配标记结果

- 纠错后数据-主题匹配.json：纠错后数据主题匹配标记结果

- 隐喻词替换后数据-主题匹配.json（示例）：隐喻词替换后数据主题匹配标记结果

- 数据扩增后训练数据集.json：经过纠错替换处理后的数据集再次扩增版本，作为模型训练原始数据集

# 面向城市规划评论的民意精准分析

## 📌 项目简介
本仓库包含论文所使用的示例数据与代码：

《面向小样本高噪声评论的民意精准洞察：多模型协同语义增强及其在城市规划评论分析微调中的验证》

本研究提出了一种多模型协同语义增强框架，用于提升小样本、高噪声场景下的民意分析效果。

---

## 📊 数据说明

由于数据来源限制，仅公开部分数据样例。

- `sample_comments.csv`：城市规划评论示例数据
- 字段说明：
  - `text`：用户评论文本
  - `label`：情感或类别标签

完整数据可在合理请求下向通讯作者获取。
